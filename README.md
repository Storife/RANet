# Code
## 1. Requirement
Pytorch (tested on 1.0.1 and 0.4.1)

torchvision = 0.2

## 2. Usage
1. Download the pretained model from this page.
2. Link DAVIS folder into datasets folder. (Please download [DAVIS 2017 version](https://davischallenge.org/davis2017/code.html).)
3. Run RANet.py
